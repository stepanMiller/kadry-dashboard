# Кадровый дашборд

Репозиторий для кадрового мониторинга. На GitHub Pages опубликована утверждённая версия дашборда от 08.07.2026 с логикой двойной принадлежности: сотрудник сохраняется в численности своего заместителя и при необходимости одновременно отображается в проектном блоке ЦСТ без двойного учёта в общих итогах.

Основная страница: `index.html`.
